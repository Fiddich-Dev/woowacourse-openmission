# woowacourse-openmission

## 미션 내용
- 달러 기준 쿠폰과 결제할인을 고려한 최저가 찾아주기
- 다른 언어로 결제 기능만들기 (비동기 처리)


### 입력
- 잔액 입력
- 구매할 상품의 이름과 가격 입력(횟수 제한)
- 결제 여부 입력



### 출력
- 현재 사용가능한 쿠폰과 결제할인 출력
- 현재 환율 출력
- 상품별 정가대비 할인가, 할인율 출력
- 결제 성공여부 출력


```
현재 잔액을 입력하세요.(원화)
345000

현재 달러 가격
1455.50원

구매하고 싶은 상품의 개수를 입력해주세요.
4

구매하고 싶은 상품의 이름과 가격(달러)을 소수점 둘째자리까지 입력해주세요.
미니pc 135.03
태블릿pc 278.00
스피커 123.49
청소기 86.18

청소기, 스피커 209.67 -> 170.00 (할인율: ??.??%)
미니pc 135.03 -> 115.78 (할인율: ??.??%)
태블릿pc 278.00 -> 189.00 (할인율: ??.??%)

청소기, 스피커 170.00 달러에 구매하시겠습니까? (yes / no)
yes
청소기, 스피커 결제 성공
미니pc 115.78 달러에 구매하시겠습니까? (yes / no)
no
태블릿pc 115.78 달러에 구매하시겠습니까? (yes / no)
yes
태블릿pc 결제 실패(잔액 부족)

주문완료
```
